## 遊戲簡介

![ULVBE_logo](https://i.imgur.com/WST3qsC.jpeg "ULVBE logo")  
#### Unlight -The Visual Basic Edition-  

一個因突發奇想而以VB6語法寫成的一套單機小遊戲。  
本作仿製Unlight的角色戰鬥系統，您可以自由選擇角色、戰鬥的地圖、BGM、以及插入您想要的事件卡組合。~宗旨為創造自由、無限想像的戰鬥平台。~
